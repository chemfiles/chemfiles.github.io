Chemfiles

An efficient library for reading and writing chemistry files

Write your code once, use it every file formats! Chemfiles provide an unified interface for accessing the data in all the supported file formats.

Formats agnostic

The biggest strength of chemfiles is to provide an unique way to access the data stored in all the formats.

Moreover, not all formats are text-based, so chemfiles support both text and binary formats.

Easy to use

Chemfiles is built in an object-oriented fashion, with only 5 classes in the public interface.

The interface focus on simplicity, and have an extensive documentation

Multi-languages

Chemfiles is written in C++, but you can use it from all the major scientific languages:

C

Fortran

Python (2 & 3)

Julia
Rust

Intended audience

Chemfiles is a software library built for scientists how write their own analysis or simulation code.

Analysis

You can reuse the same code even if you change the underlying format.

Simulation

Chemfiles can provide IO facilities, allowing you to read and write all the supported formats.

Open-source

Chemfiles is open-source, the code is hosted on Gihtub, and is open to your contibution!

Please report any bug or feature request as a Gihtub issues.

It is distributed under the MPL-v2 licence.

Goals of Chemfiles

Focus on simplicity

Chemfiles focus on simplicity, and aims to be simple, easy to use and provide a clean and consistent interface.

Chemfiles also try to make it easy to add new formats in the code, by separating file reading and format parsing.

Run on all major platforms

Chemfiles is cross-platform and run on Linux, OS X and Windows. It is usable from all the major scientific languages.

Have good quality code

Chemfiles try to have high-quality code, using modern software engineering technics such as Unit testing

Non-goals of Chemfiles

Implement analysis code

Chemfiles will not contains any code for trajectory analysis. You can find implementation of some basic analysis algorithms and a ready-to-use command line program using Chemfiles in the chrp repository.

Implement simulation code

Chemfiles will not contain molecular dynamics, Monte-Carlo, energy minimization or quantum chemistry code.
